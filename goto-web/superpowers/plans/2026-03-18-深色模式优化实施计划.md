# 深色模式优化实施计划

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 优化深色模式配色方案，采用 GitHub 灰色系统 + 靛蓝主色，提升视觉舒适度和层次感

**Architecture:** 更新 3 个核心样式文件（theme-variables.scss、element-ui-dark.scss、_modern-lab-theme.scss），使用 GitHub 经过验证的 5 层灰色体系，保持靛蓝品牌色

**Tech Stack:** SCSS, CSS Variables, Vue 2, Element UI

---

## 文件结构

本计划将修改以下文件：

```
src/assets/styles/
├── theme-variables.scss          # 核心主题变量（Task 1）
├── element-ui-dark.scss          # Element UI 适配（Task 2）
└── draw/
    └── _modern-lab-theme.scss    # Modern Lab 主题（Task 3）
```

**不修改的文件：**
- `src/utils/theme.js` - ThemeManager 工具类
- `src/store/modules/settings.js` - Vuex 状态管理
- `src/views/goto/components/ThemeToggle.vue` - 主题切换组件
- `src/assets/styles/theme-transition.scss` - 过渡动画

- [ ] **Step 3: 更新 dark-theme-variables mixin**

找到 `@mixin dark-theme-variables` 部分（约 21-37 行），替换为：

```scss
/* 深色模式变量 mixin */
@mixin dark-theme-variables {
  // === 背景层次（5层灰色系统）===
  --primary-color: #818cf8;              // 主色（保持靛蓝）
  --primary-color-light: #1e293b;        // 主色背景
  --primary-color-btn-light: #a5b4fc;    // 按钮悬浮

  --body-color: #0d1117;                 // L1: 最深背景（GitHub）
  --sidebar-color: #010409;              // L0: 侧边栏（更深）
  --card-bg: #161b22;                    // L2: 卡片背景（GitHub）
  --input-bg: #21262d;                   // L3: 输入框背景
  --hover-bg: #30363d;                   // L4: 悬浮背景（GitHub 边框色）

  // === 文本层次 ===
  --text-main: #c9d1d9;                  // 主文本（GitHub）
  --text-secondary: #8b949e;             // 次要文本（GitHub）
  --text-tertiary: #6e7681;              // 三级文本（新增）
  --sidebar-text: #8b949e;               // 侧边栏文本

  // === 边框层次 ===
  --border-color: #30363d;               // 主边框（GitHub）
  --border-color-light: #21262d;         // 浅边框（新增）
  --border-color-strong: #484f58;        // 强边框（新增）

  // === 阴影系统 ===
  --shadow-sm: 0 0 transparent;          // 深色模式减弱阴影
  --shadow-md: 0 3px 6px rgba(0,0,0,0.3);
  --shadow-lg: 0 8px 16px rgba(0,0,0,0.4);

  --border-radius: 12px;
}
```
