# 格式转换动态示意图弹窗 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `formatConvertDialog.vue` 的“转换类型”区域右侧增加示意图图标按钮，点击后用垂直居中的弹窗展示当前转换类型对应的动态 SVG。

**Architecture:** SVG 从 `tmp` 迁移到 `goto-web/src/assets/goto/plot/format-convert`，由 Vue 组件通过 `import` 静态引入。组件内部新增轻量状态、计算属性和方法，不拆新组件，避免扩大复用面和影响范围。

**Tech Stack:** Vue 2、Element UI 2.15、SCSS scoped style、Jest 字符串回归测试、ESLint。

- [ ] **Step 3: 在主 Dialog body 内加入示意图弹窗**

Insert this block after `<ChooseHeaderInfo :params="chooseHeaderInfoParams" @confirmFun="onChooseHeaderConfirm" />` and before `</template>` of the `body` slot:

```vue

      <!-- 动态示意图弹窗：只展示说明图片，不修改主表单数据 -->
      <el-dialog
        :title="currentGuideTitle"
        :visible.sync="guideDialogVisible"
        append-to-body
        custom-class="format-guide-dialog"
        width="860px"
        top="50%"
        :close-on-click-modal="true"
        @close="closeGuideDialog"
      >
        <div class="format-guide-body">
          <el-image
            v-if="!guideImageLoadError"
            class="format-guide-image"
            :src="currentGuideImage"
            fit="contain"
            @error="onGuideImageError"
          >
            <div slot="error" class="format-guide-error">
              示意图加载失败，请稍后重试
            </div>
          </el-image>
          <div v-else class="format-guide-error">
            示意图加载失败，请稍后重试
          </div>

      </el-dialog>
```

- [ ] **Step 5: 添加当前示意图计算属性**

Inside `computed`, keep `isLongToWideDisabled()` and add the two computed properties after it:

```js
  computed: {
    isLongToWideDisabled() {
      return this.params.scenario === 'handsontable'
    },
    currentGuideTitle() {
      return this.formData.type === 'l2w' ? '长转宽动态示意图' : '宽转长动态示意图'
    },
    currentGuideImage() {
      return this.formData.type === 'l2w' ? longToWideGuide : wideToLongGuide
    }
  },
```

- [ ] **Step 6: 添加打开、关闭和图片错误方法**

Inside `methods`, insert these methods before `open()`:

```js
    openGuideDialog() {
      // 打开说明弹窗前重置图片错误状态，避免上一次加载失败影响当前类型的示意图展示。
      this.guideImageLoadError = false
      this.guideDialogVisible = true
    },
    closeGuideDialog() {
      // 说明弹窗独立于转换表单，关闭时只清理弹窗自身状态，不触发表单重置或确认逻辑。
      this.guideDialogVisible = false
      this.guideImageLoadError = false
    },
    onGuideImageError() {
      // SVG 属于静态构建资源，失败时给出明确占位，避免用户看到空白弹窗。
      this.guideImageLoadError = true
    },
```
