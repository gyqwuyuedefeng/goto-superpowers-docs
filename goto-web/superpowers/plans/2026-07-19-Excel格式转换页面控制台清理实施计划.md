# Excel 格式转换页面控制台清理实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**目标：** 消除 Excel 格式转换工具页的 `onGuideTabChange` Vue 告警，清除该组件全部业务调试 `console.log`，并保留现有异常日志与业务行为。

**架构：** 保持现有单文件组件边界，通过补齐 Element UI `tab-click` 事件处理方法修复模板与组件实例的契约缺失；日志清理只删除正常业务分支中的调试调用。使用现有 Jest 源码加载测试覆盖方法行为和日志约束，不新增依赖或测试基础设施文件。

**技术栈：** Vue 2.6.14、Element UI 2.15.14、Jest 23、ESLint 5、GitNexus。

## 全局约束

- 工作目录固定为 `/mnt/f/IdeaProjects/goto-software/goto-web`。
- 仅修改 `src/views/goto/toolsBox/type/excelFormatConvert.vue` 和 `tests/unit/views/goto/toolsBox/type/excelFormatConvert.spec.js`。
- 保留目标组件现有 4 处 `console.error`，删除全部 17 处 `console.log`。
- 不处理 SockJS/Webpack Dev Server、HMR、Chrome 扩展或浏览器内置 AI 输出。

- 修改任何现有方法前必须执行 GitNexus upstream 影响分析；若索引陈旧，先在 `goto-web` 执行 `npx gitnexus analyze`。当前索引无 embeddings，不添加 `--embeddings`。
