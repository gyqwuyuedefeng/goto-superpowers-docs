# 多图例时显示排列方式和对齐方式设计

## 背景

当分类图例数量大于 1 时，"排列方式"和"对齐方式"才有实际意义（控制多个图例的排列和对齐）。当只有一个图例时，这些控件可以隐藏。

## 目标

当勾选的图例数量大于 1 时，才显示"排列方式"和"对齐方式"控件。

## 实现方案

### 数据结构理解

- `legendList.value` 是图例项数组
- 每个图例项的 `legend.showLegend.value` 表示是否启用
- 需要统计 `showLegend.value === true` 的图例数量

### 修改文件

1. **draw/legendSettingsParam.vue**
2. **plotSetting/legendSettingsParam.vue**

### 代码实现

**新增 computed 属性**
```javascript
computed: {
  /**
   * 统计已启用的图例数量
   * @returns {number} 已启用图例数量
   */
  legendCount() {
    const legendList = this.setting.legendList || { value: [] }
    return (legendList.value || []).filter(
      item => item.legend && item.legend.showLegend && item.legend.showLegend.value
    ).length
  }
}
```

**控件条件渲染**
```vue
<!-- legendDirection: 排列方式（多个启用图例时显示） -->
<SettingForm v-if="legendCount > 1" :container="setting.legendDirection">
  ...
</SettingForm>

<!-- legendBoxJust: 对齐方式（多个启用图例时显示） -->
<SettingForm v-if="legendCount > 1" :container="setting.legendBoxJust">
  ...
</SettingForm>
```

## 非目标

- 不修改其他控件的显隐逻辑
- 不修改图例列表项本身的显示逻辑

## 验收标准

- 只有一个启用图例时，"排列方式"和"对齐方式"隐藏
- 有多个启用图例时，"排列方式"和"对齐方式"显示
- 切换图例启用状态时，控件显隐正确响应
