# 企业信息本地查询工具 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 交付一个可在 Windows 上直接运行的企业信息本地查询工具，支持从 JSON 目录导入、按唯一键增量更新、关键词搜索、地区筛选、CSV 导出和浏览器单页操作界面。

**Architecture:** 在仓库根目录新增一个独立的 Python 子项目 `company-local-search/`，避免把本工具耦合进现有无关项目。后端采用 `FastAPI + SQLite + FTS5`，按“配置与路径、数据库连接、导入服务、查询导出服务、API 路由、静态前端、启动打包”拆分；前端使用无构建静态页面，由 FastAPI 直接托管，降低 Windows 打包复杂度。

**Tech Stack:** Python 3.11, Poetry, FastAPI, sqlite3, Pydantic, pytest, SQLite FTS5, HTML/CSS/JavaScript, PyInstaller, Git

In `company-local-search/src/company_local_search/services/import_service.py`, implement:

```python
class ImportMode(str, Enum):
    REBUILD = "rebuild"
    INCREMENTAL = "incremental"


class ImportService:
    def run(self, source_dir: Path, mode: ImportMode) -> ImportResult:
        ...
```

Minimum behavior:

1. `rebuild` 先清空业务表与 FTS 再导入。
2. `incremental` 按唯一键执行 upsert。
