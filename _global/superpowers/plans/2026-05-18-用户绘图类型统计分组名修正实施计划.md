# 用户绘图类型统计分组名修正 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将管理员统计链路中的“画图类型”统一改为 `groupName`，并在后续每日统计写入时把 `app_plot_group.name` 冗余保存到 `app_user_plot_type_daily_stat.group_name`。

**Architecture:** 后端新增统计快照字段 `group_name`，写入时通过 `plot_type_id -> group_id -> group_name` 补齐；排行查询直接返回统计表中的 `groupName`，移除误用内部存储键的 `plotTypeName`。前端同步改为只消费 `groupName`，并把页面展示收敛为单一“画图类型”语义。

**Tech Stack:** Java 17、Spring Data JPA、MySQL、JUnit 5、Mockito、Vue 2、Jest、GitNexus。
