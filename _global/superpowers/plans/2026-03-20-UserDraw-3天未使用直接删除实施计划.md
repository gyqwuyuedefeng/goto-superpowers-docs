# UserDraw 3 天未使用直接删除 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增一条不依赖旧标记/通知/延迟队列链路的 `UserDraw` 直接清理任务，在外部触发时删除 `update_time` 超过 3 天的记录及其临时目录。

**Architecture:** 在 `goto/system` 中新增一个任务类和一个单条删除执行器；任务类只负责扫描候选数据、逐条执行和汇总日志，执行器只负责“删目录再删库”的单条语义。查询层通过 `UserDrawSimpleRepository` 新增独立方法，仅按 `update_time < NOW() - INTERVAL 3 DAY` 选出候选数据；测试层用 Spring Boot 集成测试覆盖主流程，用 Mockito 单测覆盖失败路径。

**Tech Stack:** Java 17, Spring Boot 3, Spring Data JPA, Lombok, JUnit 5, Mockito, Apache Commons IO, GitNexus CLI/MCP

## Test Strategy

- 先写集成测试锁定 3 个核心行为：过期即删、目录缺失继续删库、未过期不删。

@Service

public class LongTimeUnusedUserDrawRemoveTask {

    private final ComponentBean componentBean;
    private final UserDrawRemoveExecutor userDrawRemoveExecutor;

        log.info("开始执行 UserDraw 直接清理任务");
        List<UserDrawSimple> candidates = componentBean.getRepository()
                .getUserDrawSimpleRepository()
                .findUnusedProjects(UNUSED_DAYS);
        int success = 0;
        int failed = 0;
        for (UserDrawSimple userDraw : candidates) {
            UserDrawRemoveExecutor.Result result = userDrawRemoveExecutor.execute(userDraw);
            if (result.isSuccess()) {
                success++;
            } else {
                failed++;
            }
        }
        log.info("UserDraw 直接清理任务完成，扫描：{}，成功：{}，失败：{}",
                candidates.size(), success, failed);
    }
}

Create `UserDrawRemoveExecutor` with a minimal passing implementation:

```java
@Slf4j
@Service
@RequiredArgsConstructor
public class UserDrawRemoveExecutor {
    private final ComponentBean componentBean;

    public Result execute(UserDrawSimple userDraw) {
        try {
            removeFolderIfNecessary(userDraw.getTmpFolderName());
        } catch (Exception e) {
            log.error("UserDraw 目录删除失败，id：{}，tmpFolderName：{}",
                    userDraw.getId(), userDraw.getTmpFolderName(), e);
            return Result.failed("FOLDER");
        }
        try {
            componentBean.getRepository().getUserDrawRepository().deleteById(userDraw.getId());
            return Result.success();
        } catch (Exception e) {
            log.error("UserDraw 数据库删除失败，id：{}，tmpFolderName：{}",
                    userDraw.getId(), userDraw.getTmpFolderName(), e);
            return Result.failed("DB");
        }
    }

    void removeFolderIfNecessary(String tmpFolderName) throws IOException {
        File folder = UserDrawUtils.obtainTmpFolder(
                componentBean.getProperties().getConfigProperties(), tmpFolderName);
        if (folder.exists() && FileUtils.isDirectory(folder)) {
            FileUtils.forceDelete(folder);
        }
    }

    @Getter
    @AllArgsConstructor(staticName = "of")
    static class Result {
        private final boolean success;
        private final String stage;

        static Result success() {
            return of(true, "NONE");
        }

        static Result failed(String stage) {
            return of(false, stage);
        }
    }
}
```

@ExtendWith(MockitoExtension.class)
class UserDrawRemoveExecutorTest {
    @Mock
    private ComponentBean componentBean;
    @Mock
    private Repository repository;
    @Mock
    private UserDrawRepository userDrawRepository;

    @BeforeEach
    void setUp() {
        when(componentBean.getRepository()).thenReturn(repository);
        when(repository.getUserDrawRepository()).thenReturn(userDrawRepository);
    }

    @Test
    void executeReturnsFailureAndSkipsDeleteByIdWhenFolderDeletionFails() throws Exception {
        UserDrawSimple userDraw = new UserDrawSimple();
        userDraw.setId(1L);
        userDraw.setTmpFolderName("tmp-a");

        UserDrawRemoveExecutor executor = Mockito.spy(new UserDrawRemoveExecutor(componentBean));
        doThrow(new IOException("boom")).when(executor).removeFolderIfNecessary("tmp-a");

        UserDrawRemoveExecutor.Result result = executor.execute(userDraw);

        Assertions.assertFalse(result.isSuccess());
        Assertions.assertEquals("FOLDER", result.getStage());
        verify(userDrawRepository, never()).deleteById(anyLong());
    }

    @Test
    void executeReturnsFailureWhenDeleteByIdThrows() throws Exception {
        UserDrawSimple userDraw = new UserDrawSimple();
        userDraw.setId(2L);
        userDraw.setTmpFolderName("tmp-b");

        UserDrawRemoveExecutor executor = Mockito.spy(new UserDrawRemoveExecutor(componentBean));
        doNothing().when(executor).removeFolderIfNecessary("tmp-b");
        doThrow(new RuntimeException("db down")).when(userDrawRepository).deleteById(2L);

        UserDrawRemoveExecutor.Result result = executor.execute(userDraw);

        Assertions.assertFalse(result.isSuccess());
        Assertions.assertEquals("DB", result.getStage());
        verify(userDrawRepository).deleteById(2L);
    }
}
