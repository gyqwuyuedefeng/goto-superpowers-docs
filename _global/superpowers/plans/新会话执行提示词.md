### Spring BeanPostProcessor 参考

```java
@Component
public class StreamListenerBeanPostProcessor implements BeanPostProcessor {

    @Override
    public Object postProcessAfterInitialization(Object bean, String beanName) {
        // 1. 获取 bean 的 Class 对象
        Class<?> clazz = bean.getClass();

        // 2. 遍历所有方法
        for (Method method : clazz.getDeclaredMethods()) {
            // 3. 检查是否有 @StreamListener 注解
            StreamListener annotation = method.getAnnotation(StreamListener.class);
            if (annotation != null) {
                // 4. 提取注解配置
                String stream = annotation.stream();
                String group = annotation.group();

                // 5. 创建端点并注册
                StreamListenerEndpoint endpoint = new StreamListenerEndpoint(
                    bean, method, annotation
                );
                streamConsumerRegistry.register(endpoint);
            }
        }

        return bean;
    }
}
```

### Q3: 如何处理并发消费？

根据 `concurrency` 配置创建多个消费线程：

```java
int concurrency = annotation.concurrency();
for (int i = 0; i < concurrency; i++) {
    String consumerName = annotation.consumerName() + "-" + i;
    startConsumerThread(endpoint, consumerName);
}
```
